INSTRUCTIONS
- This application exclusively caters to the needs for sending
    automatically generated messages to Clients.
- Please make sure, to use the Excel file given along with the
    application to be used for inputing data into the application.
- Make sure that there are no headings in the excel file while
    entering the data.
- Once you click "Send", do not have any mouse or keyboard inputs.
- If the messages are not sent, but it shows on the application
    that it has been sent, do contact the developer for changes
    in the software to suite to your needs.
- This is just a Beta Version of the application.
